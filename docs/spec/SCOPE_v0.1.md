# Scope v0.1

## In Scope
- Tool Registry (registered tools only)
- Policy Engine: embedded OPA (Rego)
- Evidence Engine: append-only log + hash chain + validate
- Adapter: MCP (required)

## Out of Scope
- RBAC/users/SSO
- Approvals/workflows
- Multi-tenant
- Remote evidence store
- UI
- Kubernetes-specific ops logic
- Generic shell execution
